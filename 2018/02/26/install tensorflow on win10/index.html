1 Install CUDA and cuDNN

Download CUDA Toolkit and cuDNN. Choose the version your GPU support.

2 Install Anaconda

2.1 Download and install Anaconda

Download Anaconda and tick “Add Anaconda to my PATh environment variable“ during installing.

2.2 Switch the software source of pip and conda

Domestic mirrors can accelerate the download speed of installing packages using pip and conda.
There are some mirrors in China for both pip and conda.

2.2.1 Pip

Pip source:

1. 阿里云 http://mirrors.aliyun.com/pypi/simple/
